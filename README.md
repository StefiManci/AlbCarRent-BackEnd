AlbCarRent Backend

Backend API for AlbCarRent, a car rental platform where car rental businesses can create profiles, add their vehicles, and manage bookings, while customers can browse cars and make reservations.

This backend powers the AlbCarRent platform by providing secure APIs for authentication, vehicle management, and rental reservations.

-Features
*User Features
 1)User registration and login

 2)Browse available cars

 3)View car details

 4)Book vehicles

5)View booking history

-Business Features

1)Create and manage business profiles

2)Add, update, and delete cars

3)Manage car availability

4)View and manage reservations

5)Track rental history

-Security

1)JWT authentication

2)Role-based access control

3)Protected endpoints for businesses

-System Overview

*The backend follows a RESTful architecture and supports multiple user roles:

1)Customer – rents cars

2)Business – manages vehicles and bookings

3)Admin – platform management

-Tech Stack

Backend: ASP .NET

Database: MySQL

ORM: Prisma / Sequelize / Mongoose

API Docs: Swagger / OpenAPI

Containerization: Docker
